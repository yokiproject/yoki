我們的首席顧問醫生

陳思昕牙科醫生（Dr. Yoki Chan）是牙科中心的首席顧問醫生，專責與患者一起制定治療解決方案，持續跟進患者的治療進展及監督牙科中心的整體醫療質素。

陳醫生同時在本地及海外具有牙科專業資格。陳醫生於香港大學取得牙醫學士榮譽學位，其後獲香港牙科醫學院以及澳洲及紐西蘭皇家牙科醫學院院員(牙科全科)頒授專業資格。
